# ☸ Kubernetes Hetzner Cloud csi-driver

Welcome to the Kubernetes documentation for the **csi-driver**.

This driver follows the CSI spec, so it works with other orchestrators like Nomad and Docker Swarm. Kubernetes is the only officially supported platform, which is why we keep one main set of docs for it.

This documentation is written and organized following the [Diátaxis guidelines](https://diataxis.fr/). Below you can find a high-level overview of where to look for certain things:

- [Guides](guides)
- [Tutorials](tutorials)
- [Reference](reference)
- [Explanation](explanation)
